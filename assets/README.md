# Assets Folder

## Google Logo Setup

To add the Google logo for the sign-in button:

### Option 1: Download Official Google Logo

1. Download the Google logo from: https://developers.google.com/identity/branding-guidelines
2. Save it as `google_logo.png` in this folder
3. Recommended size: 24x24 pixels or 48x48 pixels (for 2x)

### Option 2: Use a Free Icon

1. Visit https://iconduck.com or https://icons8.com
2. Search for "Google logo"
3. Download as PNG (24x24 or 48x48)
4. Save as `google_logo.png` in this folder

### Note

If you don't add a logo, the app will fallback to showing a login icon instead.
